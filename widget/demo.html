🤖 RAG Chatbot – Widget Demo

Embeddable AI Chat Widget

This page demonstrates how to embed the RAG-powered chatbot on any website using a single <script> tag. Click the purple button in the bottom-right corner to try it.

Architecture

How it works

LayerTechnologyRole
Knowledge Engine Python · BeautifulSoup · LangChain · ChromaDB Crawls the target website, chunks text, stores vector embeddings.
RAG Chain OpenAI Embeddings + GPT-4o-mini Retrieves relevant chunks and generates grounded answers.
Backend API FastAPI (Python) Exposes /ingest and /chat REST endpoints.
Chat Widget Vanilla JavaScript Floating UI injected into any page via a single script tag.
Integration

Embed on your website

Paste these two tags just before </body>:

<!-- 1. Configure the widget -->
<script>
  window.ChatWidgetConfig = {
    apiUrl:       "https://your-api-server.com",
    title:        "Support Chat",
    primaryColor: "#4f46e5",
  };
</script>

<!-- 2. Load the widget -->
<script src="chat-widget.js" defer></script>
Quick Start

Backend setup

# 1. Install Python dependencies
cd backend
pip install -r requirements.txt

# 2. Configure environment variables
cp .env.example .env
# Edit .env and add your OPENAI_API_KEY

# 3. Ingest your website (one-time)
python ingest.py --url https://your-site.com --max-pages 50

# 4. Start the API server
uvicorn api:app --host 0.0.0.0 --port 8000 --reload
